## 🎯 Casos de Uso

| # | Caso de Uso | Descripción |
|---|-------------|-------------|
| 1 | Investigación de Empresas | Uso de Chrome DevTools MCP para investigar empresas tech en Cartagena |
| 2 | Análisis de Archivos Locales | Análisis de carpeta Downloads para identificar archivos antiguos y pesados |
| 3 | Análisis de Ideas de Negocio | Categorización de 100 ideas en clusters y generación de mapa visual |
| 4 | Diagnóstico de Macbook | Diagnóstico de rendimiento lento y optimización del sistema |
| 5 | Investigación AEO | Investigación, escritura y publicación de artículo sobre AEO |
| 6 | Análisis y Documentación | Análisis de estructura del proyecto y creación de documentación |
| 7 | Configuración de Skill | Configuración de skill frontend-design de Anthropic |

## 🚀 Herramientas Utilizadas

- **OpenCode**: Asistente de IA para terminal
- **Chrome DevTools MCP**: Automatización de navegador para investigación web
- **CodeSearch**: Búsqueda de código y documentación
- **WebSearch**: Búsqueda en la web
- **Bash**: Comandos del sistema
- **Skills**: Agent Skills de Anthropic

## 🛠️ Skills Configuradas

### frontend-design
- **Ubicación**: `.opencode/skills/frontend-design/SKILL.md`
- **Fuente**: [anthropics/skills](https://github.com/anthropics/skills)
- **Descripción**: Crea interfaces frontend distintivas y de grado de producción con alta calidad de diseño
- **Uso**: Construir componentes web, páginas, dashboards, React components, HTML/CSS layouts

### git-commit-formatter
- **Ubicación**: `.opencode/skills/git-commit-formatter/SKILL.md`
- **Descripción**: Formatea mensajes de commit según Conventional Commits
- **Uso**: Confirmar cambios en Git con formato estándar
- **Contexto**: Proyecto SisuLine (GitHub: alarcone1/SisuLine)

**Nota**: OpenCode busca skills en `.opencode/skills/` según su documentación oficial.
